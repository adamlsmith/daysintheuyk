# Introduction

This python script will parse your Location History from Google Takeout, reverse lookup the countries you were in on each day, and record each day where you were in the UK.

The purpose of this script is for calculating Tax Residency using the HMRC's residency calculator.

# Installation

You need to install this: https://github.com/googlemaps/google-maps-services-python

Python is obviously required

# Running

You should place your Location History.json from Google in the same directory as the script

Run ./reverse.py
